# Omni Passives

This mod automatically grants all passive abilities associated with a class archetype. For example, if a cat is assigned the Tank role, it will gain every Tank-related passive from classes such as Druid, Necromancer, Fighter, and more.

## Compatibility

- Game version: `v1.0.20622`
- Author: `NightOwl`

## What Is Included

- Fighter
- Hunter
- Mage
- Cleric
- Tank
- Thief
- Monk
- Butcher
- Druid
- Tinkerer
- Necromancer
- Psychic

## Exclusions

- Colorless/no-class passives
- Disorders and hereditary/event/mutation disorder systems
- `Glutton`
- `SneakAttack`
- `RepressedMemories`

## Basic Usage

1. Install and open `Mewtator`.
2. Place this mod folder inside your Mewtator mods directory.
3. Enable `Omni Passives` in Mewtator.
4. Start a new run to apply the class passive setup.
