# Omni Passives

This mod automatically grants all passive abilities associated with a class archetype. It keeps each cat linked to its own class pool (Tank gets Tank pool, Mage gets Mage pool, etc.) and applies upgraded passive behavior at run start.

## Compatibility

- Game version target: `v1.0.20622`
- Author: `Nightfaller`

## What Is Included

- Fighter
- Hunter
- Mage
- Medic (Cleric)
- Tank
- Thief
- Monk
- Butcher
- Druid
- Tinkerer
- Necromancer
- Psychic

## Exclusions For Stability

- Colorless/no-class passives
- Disorders and hereditary/event/mutation disorder systems
- `Glutton` (excluded to prevent an infinite consumable/fly generation loop and severe lag)
- `SneakAttack`
- `RepressedMemories`
- `Beckon`
- `DeathChill`
- `EyeCatchin`
- `LongStrider`
- `VersatileVocalist`
- `HardHead`

## Basic Usage

1. Install and open `Mewtator`.
2. Place this mod folder inside your Mewtator mods directory.
3. Enable `Omni Passives` in Mewtator.
4. Start a new run to apply the class passive setup.
